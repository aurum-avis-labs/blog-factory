# GoldCrew Voxel Hybrid Avatar Style Guide

**Style Name:** Voxel Hybrid Professional  
**Visual Theme:** Anthropomorphic bird characters in sharp business attire, rendered in a clean, modern 3D voxel style.  
**Primary Use:** Team identity visuals, agent profile icons, website scenes, explainer visuals, merch opportunities.

## 1. General Aesthetic

| Attribute | Specification |
|---|---|
| Rendering Style | 3D voxel-based, modern isometric lighting, 2.5D depth |
| Block Density | Medium-resolution voxel grid; blocky yet clean and expressive |
| Material Finish | Matte with soft shadows; warm directional lighting |
| Color Palette | Neutral body tones (based on bird species), black suit attire, white or ivory undershirts, gold accent elements |
| Size & Scale | Human-proportioned silhouettes adapted to blocky construction |
| Posture | Upright, mid-action or expressive; always dynamic and work-oriented |

## 2. Character Design: Core Features

| Feature | Standard Description |
|---|---|
| Species | Each character is based on a specific bird (e.g., hawk, swan, heron, hummingbird), matched to role personality |
| Head Design | Stylized bird head in voxel structure; expressive voxel eyes with stylized eye rings or brows |
| Beak | Block-formed, matched to species; angled subtly for personality (e.g., curved hawk beak for Mark, elegant long heron beak for Sophie) |
| Facial Expression | Fixed but expressive using shape of beak, eyes, and posture |
| Crest / Hairstyle | Each character includes a sculpted voxel feather crest or blocky “hair” silhouette for character individuality |

## 3. Outfits & Accessories

| Element | Description |
|---|---|
| Suit Type | Tailored voxel-style business suits in black or slate; segmented at joints for articulation |
| Shirt | Ivory, soft white, or pastel-toned undershirts to contrast suit |
| Feather Pin | Bright golden voxel feather worn clearly on right shoulder of all agents |
| Shoes | Simple black voxel shoes for men; women may wear sleek voxel heels for distinction |
| Props | Floating task boards, calendars, styluses, clipboards — all in voxel block form with golden UI accents |

## 4. Male Character Guidelines (e.g., Manager Mark)

| Feature | Specification |
|---|---|
| Silhouette | Broad-shouldered, square-chested, grounded stance |
| Facial Structure | Strong jawlines, confident eye positioning |
| Posture | Assertive or presenting — one hand typically holding a clipboard or pointing |
| Hair/Crest | Feather crest or block-styled hair (e.g., short swept pixel hair for Mark) |
| Suit Fit | Classic business cut with slightly broader torso-to-hip ratio |

## 5. Female Character Guidelines (e.g., Sophie the Assistant)

| Feature | Specification |
|---|---|
| Silhouette | Tapered waist, subtle chest shaping, balanced legs |
| Chest Design | Subtle voxel form built into upper torso; elegant, not exaggerated |
| Leg Shape | Longer proportions with voxel heels or wedge-style shoes |
| Facial Features | Softer jaw angles, expressive eyes with accent rings (e.g., lilac for Sophie) |
| Crest / Hair | Feathered crest styled like a blocky bob, bun, or swept-back plume |
| Posture | Graceful and task-focused — often holding a calendar, stylus, or digital clipboard in one hand, gesturing with the other |

## 6. Background & Lighting

| Attribute | Specification |
|---|---|
| Background Style | Warm neutrals or light gradient fields with soft gold voxel trails or digital sparks |
| Lighting | Editorial-style soft lighting from one direction; warm tone with ambient voxel shadows |
| Task Trails | Thin, voxel-style floating UI lines or blocks to represent activity, connection, or motion flow |

## 7. Role Clarity Cues (Optional per Character)

- **Miles (Project Manager):** Clipboard with branching tasks  
- **Sophie (Assistant):** Floating calendar tile with highlighted date  
- **Parker (Designer):** Stylus + sketchpad with UI grid blocks  
- **Richard (Researcher):** Magnifying glass block and floating data nodes  
- **Max (Marketer):** Floating social post cards or hashtag tiles  
- **Ben (Analyst):** Dashboard or chart panel hovering from wrist
